行内元素与块级元素

发表于 2016-05-03

行内元素与块级元素

Block Element 块级元素

height,width可以设置。每个块级元素默认占据一整行。

Inline Element 内联元素

height,width不可设置。宽度随自身内容的宽度变化。默认不占据一整行。

常用元素

Block element

  • address - 地址
  • blockquote - 块引用
  • center - 举中对齐块
  • dir - 目录列表
  • div - 常用块级容易，也是css layout的主要标签
  • dl - 定义列表
  • form - 交互表单
  • h1 - 大标题
  • h2 - 副标题
  • h3 - 3级标题
  • h4 - 4级标题
  • h5 - 5级标题
  • h6 - 6级标题
  • hr - 水平分隔线
  • menu - 菜单列表
  • noframes - frames可选内容，对于不支持frame的浏览器显示此区块内容
  • noscript - 可选脚本内容,对于不支持script的浏览器显示此内容
  • ol - 排序表单
  • p - 段落
  • pre - 格式化文本
  • table - 表格
  • ul - 非排序列表

Inline element

  • a - 锚点
  • abbr - 缩写
  • acronym - 首字
  • b - 粗体(不推荐)
  • big - 大字体
  • br - 换行
  • dfn - 定义字段
  • em - 强调
  • font - 字体设定(不推荐)
  • i - 斜体
  • img - 图片
  • input - 输入框
  • label - 表格标签
  • q - 短引用
  • s - 中划线(不推荐)
  • samp - 定义范例计算机代码
  • select - 项目选择
  • small - 小字体文本
  • span - 常用内联容器，定义文本内区块
  • strike - 中划线
  • strong - 粗体强调
  • sub - 下标
  • sup - 上标
  • textarea - 多行文本输入框
  • u - 下划线

发表于 2016-05-02

正则表达式

  • ^代表字符串的开头
  • $代表字符串的结尾
  • \b元字符。代表单词的开头或结尾。 \bhi\b 匹配所有的hi
  • \d匹配一位数字。 同[0-9]
  • \w匹配任意字母或者数字或者下划线或者汉字
  • \s匹配任意的空白符

以上匹配符的字母换成大写即可变成反义匹配。如\W变成匹配任意不是字幕、数字、下划线、汉字的字符

  • \B
  • \D
  • \W
  • \S
  • . 匹配除了换行符以外的任何字符。
  • *表示数量 *前面的内容可以连续重复匹配任意次数包括0次。
  • +表示数量 和*不同的是 必须匹配1次或者1次以上
  • ? 匹配0次或者1次

如果想匹配 . * + 这样的符号就必须使用转义符 \

  • {数字}连续匹配n次。 \d{4}-\d{8}匹配电话号。四位区号-八位电话号。{m，n}匹配不少次m次不多于n次。{m,}重复匹配n或者更多次
  • [ ]表示匹配集合中的某一个字符
  • ( )表示分组，相当于子表达式。
  • [^x] 匹配除了x以外的任意字符
  • (^abcd)匹配除了abcd意外的任意字符

练习：解读 ((2[0-4]\d|25[0-5]|[01]?\d\d?)\.){3}(2[0-4]\d|25[0-5]|[01]?\d\d?) 匹配IP地址的正则表达式。

<a[^>]+>匹配用尖括号括起来的以a开头的字符串。
