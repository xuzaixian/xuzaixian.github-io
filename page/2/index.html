行内元素与块级元素

Block Element 块级元素

height,width可以设置。每个块级元素默认占据一整行。也就是默认宽是容器的100%。

Inline Element 内联元素

height,width不可设置。宽度随自身内容的宽度变化。默认不占据一整行。

Display:Inline-Block

CSS中将元素设置为Display:inline-block,可以使行内元素具有一些块级元素的特点。也就是内部表现类似Block，外部表现类似Inline。

  • 可以设置宽高
  • 默认占一行。
  • IE6/7不支持Inline-Block.需要有*zoom:1来触发元素hasLayout才会表现出来。

JS中的对象

属性类型

在JS中有两种属性类型：数据属性，访问器属性。

数据属性

Configurable：能否通过delete删除属性。默认为true.
Enumerable：能否通过for-in循环返回属性。默认为true.
Writable：能否修改属性的值。默认为True.
Value：包含这个属性的数据值。读取和写入属性的时候都是从这个属性来。默认为undefined.

访问器属性

Configurable：能否通过delete删除属性。默认为true.
Enumerable：能否通过for-in循环返回属性。默认为true.
get:在读属性时调用的函数。默认为undefined.
set:在写属性时调用的函数。默认为undefined.
其中get和set不是必须的。

使用Object.getOwnPropertyDescriptor(“object”,”property”)方法取得属性的描述。
返回一个对象包含属性中的对象。

创建对象

工厂模式

构造函数模式

特征：没有显示创建Object,直接赋值给this,没有return.
内部步骤
1.创建一个新对象
2.将构造函数的作用域赋给一个新对象。this指向新对象
3.执行构造函数
4.返回新对象

原型模式

如果不是向prototype中添加方法而是指定一个prototype为一个对象，那么他的constructor属性不再指向原来的constructor需要手动重新指定。

混合模式

这种方式使得每个对象既有自己的属性又可以共享公共方法达到节约内存的目的。

稳妥构造函数模式

稳妥对象指的是没有公共属性，不引用this的对象。适合在安全环境中使用。

除了sayhi之外没有别的方法访问name的值。

正则表达式

参考链接：http://deerchao.net/tutorials/regex/regex.htm

正则表达式基本语法

  • ^代表字符串的开头
  • $代表字符串的结尾
  • \b元字符。代表单词的开头或结尾。 \bhi\b 匹配所有的hi
  • \d匹配一位数字。 同[0-9]
  • \w匹配任意字母或者数字或者下划线或者汉字
  • \s匹配任意的空白符

以上匹配符的字母换成大写即可变成反义匹配。如\W变成匹配任意不是字幕、数字、下划线、汉字的字符

  • \B
  • \D
  • \W
  • \S
  • . 匹配除了换行符以外的任何字符。
  • *表示数量 *前面的内容可以连续重复匹配任意次数包括0次。
  • +表示数量 和*不同的是 必须匹配1次或者1次以上
  • ? 匹配0次或者1次

如果想匹配 . * + 这样的符号就必须使用转义符 \

  • {数字}连续匹配n次。 \d{4}-\d{8}匹配电话号。四位区号-八位电话号。{m，n}匹配不少次m次不多于n次。{m,}重复匹配n或者更多次
  • [ ]表示匹配集合中的某一个字符
  • ( )表示分组，相当于子表达式。
  • [^x] 匹配除了x以外的任意字符
  • (^abcd)匹配除了abcd意外的任意字符

练习：解读 ((2[0-4]\d|25[0-5]|[01]?\d\d?)\.){3}(2[0-4]\d|25[0-5]|[01]?\d\d?) 匹配IP地址的正则表达式。

<a[^>]+>匹配用尖括号括起来的以a开头的字符串。

JS中的RegExp类型

新建RegExp实例

pattern是上一节的语法所表示的正则表达式。flags相当于正则表达式的使用参数.有如下几种

  • g 表示全局模式，表示即使发现了第一个匹配结果也将继续匹配直到匹配完整个字符串。
  • i 表示不区分大小写。
  • m 表示多行模式 即使遇到换行的时候也将继续匹配直到匹配完整个字符串

RegExp实例属性

  • global bool类型 表示是否设置了g标志
  • ignoreCase bool类型 表示是否设置了i标志
  • lastIndex number类型 表示开始搜索下一个匹配项的字符位置，从0开始。
  • multiline bool类型 表示是否设置了m标志
  • source 正则表达式的字符串表示。

RegExp实例方法

  • exec(str) 最主要的方法，str参数是要匹配的字符串，没有匹配项时返回null,如果成功匹配，则返回一个有input和index属性的数组.
    input表示应用正则表达式的字符串。
    index表示匹配项在字符串中的位置。
  • test(str)，str参数是要匹配的字符串，在模式与参数匹配的情况下返回true,否则false。

RegExp与String共同使用的方法

search

String.search(reg).搜索符合规则的内容，搜索失败返回-1，成功则返回出现的位置。

match

String.match(reg).搜索符合规则的内容，成功返回所有匹配成功的内容组成的数组，失败返回null。

replace

String.replace(reg,newStr|callback):newStr用来替换匹配成功的内容。或者第二参数传入一个回掉函数，该函数接受一个参数，表示成功匹配到的内容。

split

String.split(separator[,limit]),把字符串分割成一个字符串数组，reg参数可以是正则表达式或者字符串。
